# ADR-003：Affinity 系統（1-12 一維三錨點）

## 狀態

**已接受（Accepted）**

## 背景

InfoLit 遊戲需要讓「演員性格」與「騙術方式」自然對應，同時保留足夠多樣性讓每局遊戲不重複。

## 決策

**一維數值 1–12，三個錨點**：

```
1──────────3──────────6──────────9──────────12
        情感訴求    邏輯操弄   正式權威    魅力/超然型
        (pathos)   (logos)    (ethos)   （超出典型權威）
```

- **錨點 3（情感訴求）**：恐懼、煽情、愛心包裝
- **錨點 6（邏輯操弄）**：斷章取義、假邏輯、以偏概全
- **錨點 9（正式權威）**：偽造引用、假借頭銜、名牌加持
- **10–12（魅力/超然型）**：10 怪咖偽學者、11 純氣場魅力、12 神棍訴諸神秘感

**加權公式**：

```rust
weight = 1.0 / (1.0 + |actor.affinity - pattern.affinity|)
```

**15% 例外通道**：完全隨機，確保多樣性、防 metagame。

## 為何選 1-12 而非 1-9

- 9 作為「正式權威」錨點若在末端，10+ 的「氣場壓制」「神棍型」角色無法表達
- 10–12 讓這類角色有語義空間
- 等距三錨點（3/6/9）讓分類更直覺

## 內容作者指引

| 要表達... | affinity 值 |
|---|---|
| 純情感煽動（恐懼、愛心） | 1–3 |
| 邏輯操弄（斷章取義、以偏概全） | 5–7 |
| 引用/頭銜型權威 | 8–10 |
| 純氣場/神秘感 | 10–12 |
